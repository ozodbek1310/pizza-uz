# 🍕 Pizza UZ

A "Pizza & Burger House" restaurant landing page built with HTML and CSS.

🔗 **Live demo:** https://pizza-uz-delivery.netlify.app/

## 📖 About

A simple restaurant website showing the pizza and burger menu with prices, delivery info, and an order button.

## ✨ Features

- Menu sections for pizzas and burgers, each with name, description, and price
- Delivery info section (free delivery over $25, estimated delivery time)
- "Order Now" call-to-action
- Contact details in the footer
- Responsive, clean layout

## 🚀 Getting Started

To run this project locally:

1. Clone the repo
   ```bash
   git clone https://github.com/ozodbek1310/pizza-uz.git
   ```
2. Open `index.html` in your browser

No build tools or dependencies needed — it's plain HTML & CSS.
